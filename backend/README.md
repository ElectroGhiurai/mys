# mys-backend

## Installation 

OpenJDK 17 or higher is required to run the application -> https://download.oracle.com/java/26/latest/jdk-26_windows-x64_bin.exe

VS Code Extensions:
1. Extension Pack for Java
2. Spring Boot Extension Pack
3. Spring Initializr Java Support

Maven is required to build the application. You can install it from [Maven's official website](https://maven.apache.org/install.html). or do this -> choco install maven

check in terminal that these are installed
java -version
mvn -version

then run server with 
mvn spring-boot:run
